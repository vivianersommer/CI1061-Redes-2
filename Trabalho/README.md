# CI1061-Redes 2

Compilar:
````
make
````

Rodar trabalho - cliente:
````
./cliente <nome-do-servidor> <porta> 
````

Exemplo - cliente:
````
./cliente <hostname> 8887
````

Rodar trabalho - servidor:
````
./servidor <porta> 
````

Exemplo - servidor:
````
./servidor 8887
````

**OBS: sempre rodar primeiro o servidor!**
